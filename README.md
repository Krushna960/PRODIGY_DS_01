# Task-01: Population Data Visualization

## Objective
The objective of this task is to visualize population data using basic data visualization techniques such as bar charts and pie charts. This helps in understanding population distribution across countries.

## Dataset
The dataset used for this task is `world_population.csv`, which contains country-wise population data.

## Visualizations Performed
- **Bar Chart:** Shows the top 10 most populated countries, enabling easy comparison of population sizes.
- **Pie Chart:** Represents the population share of the top 5 most populated countries, highlighting how population is distributed among major countries.
- **Bar Chart:** Shows the year wise population of india.

## Tools and Technologies
- Python
- Pandas
- Matplotlib

## Outcome
Through this task, basic data loading, sorting, and visualization skills were applied. The visualizations provide clear insights into global population distribution and demonstrate fundamental data analysis capabilities.
